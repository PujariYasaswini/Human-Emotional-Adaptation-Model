# Human-Emotional-Adaptation-Model

## Project Overview
The Human Emotional Adaptation Model (HEAM) is a real-time system that detects human emotions using facial expressions and voice signals. It identifies emotions like happy, sad, angry, and neutral, and provides resilience-based feedback. Built using Python, OpenCV, and machine learning techniques.

## Features
- Real-time facial emotion detection
- Voice-based emotion recognition
- Multimodal emotion analysis
- Resilience feedback mechanism
- CPU-based lightweight model

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- Librosa
- Matplotlib

## Emotions Detected
- Happy
- Sad
- Angry
- Fear
- Surprise
- Neutral
- Disgust

## Results
The system achieved stable predictions for both facial and voice emotion detection models.

## Applications
- Mental health monitoring
- Human-computer interaction
- Smart assistants
- Emotion-aware AI systems
